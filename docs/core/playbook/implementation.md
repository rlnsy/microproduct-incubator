## Build Sequence

1. Implement one core user flow end-to-end.
2. Add analytics for key action completion.
3. Release to a small pilot audience quickly.

## Definition of Done (MVP)

- Core task completed in under 3 minutes.
- Basic reliability and error handling in place.
- One feedback loop collected from real users.
